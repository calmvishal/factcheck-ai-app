# AI Fact Checker

An AI-powered fact checking web app that extracts factual claims from PDF documents and verifies them using live web search results.

## Features

- Upload PDF documents
- Extract factual claims
- Verify claims using live web data
- Detect inaccurate or false information

## Tech Stack

- Python
- Streamlit
- OpenRouter API
- DuckDuckGo Search
- PyMuPDF

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
